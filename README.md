# Template Automation Documentation

This README provides an index to the full documentation generated by `pdoc`.  
Click on any module to view its detailed documentation.

To generate the documentation in local server, run:
```bash
mkdoc serve
```
To generate the documentation in HTML format, run:
```bash
mkdoc build
```
